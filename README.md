# Crystal RNN Repo

This repository bundle contains:

- `base_classes/` with the refactored core classes and physics utilities
- `find_models.py` for training and selecting candidate models
- `coords_lmp.raw` as the local trajectory dataset

## GitHub Publishing Note

`coords_lmp.raw` is kept in this local folder but is excluded from git tracking for now.
The file is too large for a regular GitHub upload and should be handled separately before publishing.
